createBlock

  • createBlock(str: string): BlockType
    • Defined in block_compiler.ts:50

    Compiling blocks is a multi-step process:

    1. build an IntermediateTree from the HTML element. This intermediate tree is a binary tree structure that encode dynamic info sub nodes, and the path required to reach them
    2. process the tree to build a block context, which is an object that aggregate all dynamic info in a list, and also, all ref indexes.
    3. process the context to build appropriate builder/setter functions
    4. make a dynamic block class, which will efficiently collect references and create/update dynamic locations/children
